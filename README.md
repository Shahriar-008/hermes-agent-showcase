# Hermes Agent Showcase

> **This is not a chatbot setup. This is agentic infrastructure.**

A comprehensive documentation of a production-grade, multi-profile Hermes Agent deployment running 27 scheduled cron jobs across 3 isolated profiles, supporting 3 active software projects, and powering a live Telegram market intelligence channel — all from a single VPS.

---

## What's Inside

Hermes Agent is an open-source AI agent framework by [Nous Research](https://github.com/NousResearch/hermes-agent). This repository documents a real-world deployment that pushes it far beyond casual use:

| Capability | Scale |
|---|---|
| **Cron Jobs** | 27 scheduled automations |
| **Profiles** | 3 fully isolated identities |
| **Skills** | 71 across 18 categories |
| **Model Tiers** | 5-tier cost-optimized router |
| **Projects Built** | 3 (CLI tool, static site, data pipeline) |
| **Subagent Workers** | Orchestrator pattern with Claude Code |
| **Self-Monitoring** | Watchdog + audit feedback loops |
| **Data Sources** | 6+ free public APIs (no paid keys) |
